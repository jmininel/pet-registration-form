## 📋 Sobre o Projeto

O Pet Registration Form é uma aplicação web criada para realizar o cadastro de animais de estimação e seus respectivos tutores através de uma interface moderna, intuitiva e totalmente responsiva.

O projeto foi desenvolvido utilizando tecnologias atuais do ecossistema React, com foco em validação de dados, tipagem estática, acessibilidade e organização escalável de código.

Além do cadastro das informações do pet, o sistema aplica validações em tempo real para garantir maior consistência dos dados e melhor experiência para o usuário.

---

## ✨ Funcionalidades

* Cadastro completo de animais de estimação
* Cadastro de informações do tutor
* Validação avançada de formulários
* Feedback visual para campos inválidos
* Interface responsiva para desktop e dispositivos móveis
* Componentes reutilizáveis
* Tipagem segura com TypeScript

## ⚙️ Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/jmininel/pet-registration-form.git
```

### Acesse a pasta

```bash
cd pet-registration-form
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```

Abra:

```bash
http://localhost:3000
```
